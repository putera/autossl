# Automated SSL Renewal for Let's Encrypt

Requirement :
- cPanel Shared Hosting
- Telegram API Key (Notification via Telegram App)

Cronjob Setting :
- Add new cronjob schedule (1 per day at 7 AM)
- Run this command

`
	php -q /PATH-TO-YOUR-CRONJOB-SCRIPT.php >/dev/null 2>&1
`
